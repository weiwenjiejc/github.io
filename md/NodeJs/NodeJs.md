# npm管理器

> 默认全局安装位置C:\Users\wei\AppData\Roaming\npm\node_modules，C:\Users\wei\AppData\Roaming\npm

* npm install -g，全局安装
* npm install，默认下载到当前命令行所在的目录

```shell
# 安装bootstrap
npm install -g bootstrap

npm install -g jquery
```
